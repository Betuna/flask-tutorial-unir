Paso a Paso: Ejemplo de Flask

Este ejercicio se basa en el repositorio de GitHub: https://github.com/josedom24/ejemplos_flask/tree/master/ejemplo4

Objetivo

Crear una aplicación web con Flask que maneje formularios, procese datos enviados por el usuario y los muestre en una plantilla.

Paso a Paso

  1. Clonar o descargar el repositorio 
    git clone https://github.com/josedom24/ejemplos_flask.git
    o descarga el ZIP y extrae su contenido.

  2. Accede a la carpeta del ejemplo 
    cd ejemplos_flask/ejemplo4

  3. Activa tu entorno virtual (opcional pero recomendado) 
    python -m venv venv
venv\Scripts\activate

  4. Instala Requerimientos para el proyecto: 
    pip install -r requirements.txt

    5. El archivo requirements.txt se encuentra en la base del proyecto, debes copiarlo a la carpeta ejemplo4

  5. Explora los archivos principales del ejemplo:
    • app.py: código principal de la aplicación.
    • templates/formulario.html: plantilla con el formulario HTML.
    • templates/respuesta.html: plantilla para mostrar los datos enviados.
  6. Ejecuta la aplicación: 
    python app.py
    El servidor iniciará en: 
    http://127.0.0.1:5000/
  7. Prueba la aplicación:
    • Abre el navegador y visita http://127.0.0.1:5000/.
    • Rellena el formulario con tus datos y envíalo.
    • La página de respuesta mostrará los datos que enviaste.

Archivos clave

  • app.py: define las rutas / y /recibe, procesa los datos enviados desde el formulario.
  • templates/formulario.html: muestra un formulario con campos para rellenar.
  • templates/respuesta.html: muestra los datos recibidos del formulario.

Resultado

Al completar el ejercicio, tendrás una aplicación web funcional que recibe datos desde un formulario y los muestra en una página diferente. Este ejemplo es ideal para aprender a manejar formularios y rutas POST/GET en Flask.
