Instalación de Flask

¿Se precisa instalación?

  • Sí, Flask no viene con Python por defecto.
  • Se instala fácilmente con el gestor de paquetes pip.

Pasos para instalar en Windows 11 con Visual Studio Code

  1. Asegúrate de tener Python 3.7+ instalado.
  2. Verifica que pip funcione: 
    python --version
pip --version
  3. (Opcional) Crea un entorno virtual para tu proyecto: 
    python -m venv venv
    y actívalo: 
    venv\Scripts\activate
  4. Instala Flask en el entorno: 
    pip install Flask

Versiones necesarias

  • Python: mínimo 3.7, recomendado 3.11.
  • Flask: usa la última versión estable.

Rutas a tener en cuenta

  • Trabaja siempre desde la carpeta raíz de tu proyecto para que las rutas relativas funcionen.
  • Guarda tus archivos .py en la raíz, y las plantillas en una carpeta llamada templates/.

Variables de entorno (opcionales pero recomendadas)

Define estas variables para desarrollo: 
set FLASK_APP=app.py
set FLASK_ENV=development

Puedes agregarlas en tu terminal antes de ejecutar o configurarlas en tu .env.

¿Necesita base de datos?

  • No es obligatoria.
  • Pero puedes integrarlo con bases como SQLite, MySQL o PostgreSQL cuando tu proyecto lo requiera.

Otros consejos

  • Instala también un editor como Visual Studio Code para escribir tu código cómodamente.
  • Instala la extensión Python en VS Code para resaltar y depurar el código.
  • Mantén tu entorno virtual para evitar conflictos con otros proyectos.
