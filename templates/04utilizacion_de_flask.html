Utilización del framework Flask

¿Cómo empezar a crear una aplicación base?

  1. Crea una carpeta para tu proyecto.
  2. Dentro de esa carpeta, crea un archivo app.py con la estructura básica de Flask: 
    
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "¡Bienvenido a tu aplicación Flask!"

if __name__ == "__main__":
    app.run(debug=True)
  3. Crea una carpeta templates/ para tus archivos HTML y una carpeta static/ para tus archivos CSS y JS.
  4. (Opcional) Crea un entorno virtual y activa antes de instalar Flask y ejecutar el servidor.

¿Cómo visualizar los resultados de la ejecución?

  1. Abre una terminal en la carpeta del proyecto.
  2. Activa tu entorno virtual si lo tienes.
  3. Ejecuta el archivo app.py: 
    python app.py
  4. Verás un mensaje indicando que el servidor está corriendo en una dirección, normalmente: 
    * Running on http://127.0.0.1:5000/
  5. Abre tu navegador y visita esa dirección para ver tu aplicación en funcionamiento.
  6. Cualquier cambio que hagas en los archivos se reflejará al recargar la página (si tienes debug=True).
